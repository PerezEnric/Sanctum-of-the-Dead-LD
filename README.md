
# Sanctum of the Dead

One page dungeon made in RPG Maker for the Level Design Subject in CITM (Terrassa).

Turn-Based RPG where the player is requested to defeat a Cult based on a Sacntum. 
The player will have to kill them all so he/she will complete the quest.

 # Controls

- W/A/S/D or left mouse click to move
- Interact with left mouse click
